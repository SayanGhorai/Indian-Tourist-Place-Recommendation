# Indian Tourist Place Recommendation

Hybrid TF-IDF + SBERT-based place recommender for Indian tourist places.

## Quick start

1. Put the raw dataset `Review_db.csv` into `data/` (local only; do NOT push data to GitHub).
2. Create a Python venv and install dependencies:
   ```bash
   python -m venv venv
   # Windows
   .\venv\Scripts\activate
   pip install -r requirements.txt
   ```
